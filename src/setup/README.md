# Setup

This section covers getting your development environment ready for GPUI development.

## Topics

- [Development Environment](./dev-environment.md) - Tools and dependencies
- [Your First GPUI App](./first-app.md) - Building hello world from scratch

## Prerequisites

Before starting with GPUI, you should have:

- Rust toolchain installed (rustup recommended)
- Basic familiarity with Rust
- A code editor with Rust support

## Quick Start

```bash
cargo new my-gpui-app
cd my-gpui-app
cargo add gpui
```

See [Your First GPUI App](./first-app.md) for a complete walkthrough.
